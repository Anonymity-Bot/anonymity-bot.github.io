Setup

  1. Click on this link to add Anonymity Bot to your server.
  2. Use the /anon-register-channel command to register channels.
  3. Thats it. Now you can use the /anon command to send messages anonymously.

Note:

  • You can also use the /anon-unregister-channel command to unregister a channel.
  • You need to have the Manage Channels permission to use the /anon-register-channel and /anon-unregister-channel commands.

Features

  • Sends anonymous messages in a very pretty and natural format.
  • Supports attachments.
  • Supports sending anonymous messages from any channel to any channel.
  • Allows moderators to set if attachments are allowed in a particular channel.
  • Allows moderators to set if /anon-regenerate(see below) is allowed in a particular channel.

How to use

Just use the /anon slash command.

Its that easy.



How to change your anonymous profile

Just use the /anon-regenerate slash command.

Its that easy.



Spam prevention mechanisms

  • No @mention will ping anyone. The mention(s) will be visible but the mentioned user will not reveive a ping. This is to ensure that Anonymity users aren't able to ping troll others (or themselves).
